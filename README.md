# soc-log-analyzer-python
Python-based log analyzer to detect suspicious activities
 Python Log Analyzer & Intrusion Detection System

 Description
This project analyzes system logs and detects suspicious activities like brute-force attacks and unusual login behavior.

 Features
- Detects multiple failed login attempts
- Identifies suspicious login times
- GUI-based interface

Technologies
- Python
- Regex
- Tkinter

 Use Case
Simulates real-world SOC Analyst work.
